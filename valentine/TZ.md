# Техническое задание: Сайт-приглашение на День Святого Валентина

## 1. Цветовая схема
- **Основной цвет**: Розовый, принцесскин цвет
- **Тон**: Очень нежный и приятный
- **Палитра**: Мягкие розовые оттенки (pastel pink)

## 2. Тематика сайта
- **Основная тема**: Приглашение стать моей валентинкой
- **Стиль**: Романтичный, нежный, милый

## 3. Функционал кнопок

### 3.1 Кнопка "Да"
- При нажатии на кнопку "Да":
  - По центру экрана появляется текст: **"Thank you, i love you!"**
  - Текст отображается крупно и заметно

### 3.2 Кнопка "Нет"
- **Основное правило**: Нажать на кнопку "Нет" не получится
- При попытке нажать на кнопку "Нет" активируется одна из 7 различных анимаций (случайным образом или по очереди):

  1. **Улетание**: Кнопка улетает в другую часть экрана
  2. **Увеличение текста**: Кнопка становится жирной и увеличивается на весь экран с текстом "Кнопка нет не принимается"
  3. **Сгорание**: Анимация, что кнопка "сгорает" (эффект огня/исчезновения)
  4. **Вращение**: Кнопка начинает быстро вращаться и улетает
  5. **Дрожание**: Кнопка сильно дрожит и трясется
  6. **Уменьшение**: Кнопка уменьшается до микроскопического размера
  7. **Размытие**: Кнопка становится размытой и невидимой

- После каждой анимации кнопка возвращается на место или появляется в новом месте

## 4. Изображения
- Все изображения из папки `images/` должны быть размещены на странице
- **Расположение**: Разбросаны по всему экрану случайным образом
- **Поворот**: Каждое изображение имеет случайный наклон (разные углы поворота)
- **Размер**: Изображения могут быть разных размеров для визуального интереса
- **Позиционирование**: Случайное позиционирование по всей области экрана

## 5. Адаптивность
- **Мобильные устройства**: Сайт должен корректно отображаться на телефонах
- **Планшеты**: Оптимизация для планшетов
- **Десктоп**: Полноценное отображение на ПК
- **Responsive design**: Использование медиа-запросов и гибкой верстки

## 6. Технические требования
- **Технологии**: HTML, CSS, JavaScript (без фреймворков)
- **Совместимость**: Современные браузеры (Chrome, Firefox, Safari, Edge)
- **Производительность**: Плавные анимации без лагов

## 7. Структура файлов
```
valentine/
├── index.html
├── style.css
├── script.js
├── images/
│   ├── (все изображения пользователя)
└── README.md
```

## 8. Дополнительные требования
- Плавные переходы и анимации
- Приятная типографика
- Интуитивно понятный интерфейс
- Эмоциональное воздействие через визуальные эффекты
